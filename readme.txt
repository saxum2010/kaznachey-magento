Для того чтобы установить модуль оплаты www.kaznachey.ua проделайте следующие шаги:

1) Скопируйте файлы "kaznachey.processing.php", "kaznachey.success.php" и папку "app" в основную директорию вашего магазина.

2) Зайдите в админ-панель вашего магазина -> Система -> Конфигурация -> Продажи -> Методы оплаты

3) Заполните поля настроек пункта Kaznachey в соответствии с вашими требованиями. Идентификатор продавца и Секретный ключ можно получить в личном кабинете на www.kaznachey.ua
